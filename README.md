# 日记类小程序

使用小程序云开发的 AI 日记类小程序。

通过免费调用腾讯 NLP 情感识别，可以识别到用户记录下日记的情绪值并给予情绪币的奖励。

情绪币可以用于打捞别人公开的日记。

主要功能有：情绪排行，写日记，打捞日记。
